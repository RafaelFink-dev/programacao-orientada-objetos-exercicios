## Exercícios – Programação Orientada a Objetos

Este repositório contém exercícios e exemplos desenvolvidos durante a disciplina **Programação Orientada a Objetos** da **Pós-Graduação em Desenvolvimento Full Stack**.

### Objetivo

Registrar as atividades práticas realizadas ao longo da disciplina, com foco na compreensão dos conceitos fundamentais da Programação Orientada a Objetos, aplicados por meio de exemplos e exercícios desenvolvidos em aula.

### Conteúdos abordados

- Introdução à Programação Orientada a Objetos  
- Criação e utilização de classes e objetos  
- Definição de atributos e métodos  
- Instanciação de objetos  
- Organização básica do código  
- Exercícios práticos para fixação dos conceitos apresentados em aula  

### Tecnologias utilizadas

- JavaScript  

### Contexto acadêmico

Os exercícios apresentados foram desenvolvidos como parte das atividades da disciplina **Programação Orientada a Objetos**, no contexto da **Pós-Graduação em Desenvolvimento Full Stack**, sendo utilizados como evidência prática de aprendizagem no Memorial Descritivo do Trabalho de Conclusão de Curso (TCC).
